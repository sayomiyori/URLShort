# Скриншоты Grafana для README

Сохраните сюда PNG после нагрузочного прогона (Grafana → панель → меню «⋯» → **Share** → **Download image**).

Рекомендуемые имена файлов (используются в корневом `README.md`):

| Файл | Панель дашборда URLShort |
|------|---------------------------|
| `grafana-rps.png` | Redirect RPS |
| `grafana-latency.png` | Redirect latency (p50/p95/p99) |
| `grafana-cache-ratio.png` | Cache hit ratio |
| `grafana-rate-limit.png` | Rate limit rejections / min |
| `grafana-active-urls.png` | Active URLs |
| `grafana-overview.png` | Полный дашборд (опционально) |

После добавления файлов в корневом `README.md` раскомментируйте блок «Графики» или проверьте, что ссылки на картинки совпадают с именами выше.
